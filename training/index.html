IaaS Orchestration 101

Trainees will get a guided tour of IaaS orchestration API templating systems (e.g. AWS CloudFormation), including a high-level overview of the lower-level APIs they provide access to.

  • Orchestration API Overview

  • An introduction to Resources

  • Custom resources

  • Effective use of Orchestration APIs
Building Blocks

Trainees will learn how to leverage the four primary primitives provided by SparkleFormation, and a close look at the SparkleFormation CLI tool (sfn).

  • Components

  • Dynamics

  • Registry

  • Templates

  • sfn walkthrough
Practical Application

Trainees will be taught how to apply what they've learned using best-practices to compose example stacks for popular application architectures and services, including:

  • Infrastructure architecture best practices

  • The "Hello World" of application stacks

  • VPC creation and discovery

  • Isolating resources and nested stacks
Workshops

Trainees will spend an entire day in a series of workshops, covering advanced topics and receiving hands-on assistance from trainers, including:

  • Defining and triggering Configuration Management

  • SparkleFormation infrastructure development workflows

  • Reusability and organizational standards enforcement

  • Extending capabilities with Custom Resources
Professional Services

On-site training comes with 2-days of Professional Services provided by one of BLAH's Senior Operations Engineers. Professional services will be custom-tailored to the unique needs of the Customer (e.g. bootstrapping migration to SparkleFormation from native Orchestration API templates, assessment and enhancement of existing SparkleFormation templates, etc).
